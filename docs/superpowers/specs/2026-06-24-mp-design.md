# 微信小程序互动阅读 — Design Spec

## Architecture
Backend (Express + sql.js) provides REST API. Uni-app project compiles to WeChat Mini Program.

## Database (new tables in existing sql.js)
- mp_users: openid, nickname, avatar
- comments: article_id, user_id, content
- likes: article_id, user_id (PK)
- bookmarks: article_id, user_id (PK)

## API Changes
- New route /api/mp/* with 7 endpoints (login, comments CRUD, likes toggle, bookmarks)
- Public article APIs augmented with like_count, comment_count
- Mini program JWT auth middleware (separate from admin auth)

## Mini Program Pages (Uni-app)
- Tab: Home (article feed), Categories, Bookmarks, Profile
- Push: Article Detail (content + likes + comments + bookmark)
- Auto-login on startup via WeChat code
